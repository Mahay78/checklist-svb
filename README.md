# Checklist SVB - Inspección de Ambulancia Tipo B

Aplicación web progresiva (PWA) para la inspección mensual de ambulancias de Soporte Vital Básico (SVB) Clase B del Servicio de Urgencias Canario.

## Características

- ✅ Checklist completo con +460 artículos en 72 secciones
- ✅ 100% offline (Service Worker + localStorage)
- ✅ Exportación a PDF (Acta oficial + Lista de pedido)
- ✅ Exportación a CSV compatible con Excel
- ✅ Backup y restauración en JSON
- ✅ Filtros: Pendientes, No Cumplen, Caducados, Observaciones, A Pedir
- ✅ Modo "A Pedir" con selección múltiple de materiales a reponer
- ✅ Navegación por secciones con buscador
- ✅ Modo oscuro/claro
- ✅ Instalable en móvil como app (PWA)
- ✅ Diseño responsive

## Uso

1. Abrir `https://mahay78.github.io/checklist-svb/` en Chrome
2. Pulsar "Instalar aplicación" desde el menú ⋮ para uso sin conexión
3. Rellenar los datos del vehículo y el checklist
4. Exportar acta PDF o lista de pedido

## Tecnologías

- HTML5 + CSS3 + JavaScript (Vanilla)
- Service Worker (caché offline)
- localStorage (persistencia de datos)
- GitHub Pages (hosting)
